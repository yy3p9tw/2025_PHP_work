API 端點說明

支援參數：
category_id: 分類ID篩選
page: 頁碼 (預設: 1)
limit: 每頁數量 (預設: 12, 最大: 50)
price_min: 最低價格
price_max: 最高價格
search: 搜尋關鍵字
sort: 排序方式 (newest, oldest, price-asc, price-desc, name-asc, name-desc)
回傳格式：
{
  "success": true,
  "products": [...],
  "pagination": {...},
  "category": {...},
  "filters": {...}
}

支援參數：
id: 產品ID (必須)
回傳格式：
{
  "success": true,
  "product": {
    "id": 1,
    "name": "產品名稱",
    "categories": [...]
  },
  "related_products": [...]
}

回傳樹狀結構的分類資料，包含父子關係。

回傳格式：
{
  "success": true,
  "categories": [
    {
      "id": 1,
      "name": "動漫公仔",
      "children": [...]
    }
  ]
}

功能展示示例

分類瀏覽流程
  1. 用戶點擊導覽列「商品分類」
  2. 選擇特定分類（如：火影忍者）
  3. 跳轉到產品列表頁面
  4. 顯示該分類下的所有產品
  5. 可進一步篩選和排序
產品搜尋流程
  1. 用戶在產品頁面輸入關鍵字
  2. 系統即時搜尋相關產品
  3. 顯示符合條件的產品列表
  4. 可搭配分類和價格篩選
  5. 點擊產品查看詳細資訊
